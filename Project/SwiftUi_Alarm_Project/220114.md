### Passing Data To VM 
- VM Class에서 Model값을 초기화 할 때, a정확한 인덱스를 넘겨주는 프로퍼티가 존재.
- 이때 인덱스 프로퍼티는 이전 스크린에서 전달받은 값으로 초기화됨.
```swift
 // 이전 화면

 }
```
